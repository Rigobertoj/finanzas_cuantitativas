# Course Notes

Esta carpeta contiene el material historico del curso: notebooks, notas,
outputs, PDFs y proyectos exploratorios. Su proposito es conservar el contexto
academico sin mezclarlo con el codigo reusable de `src/modulos`.

## Estructura

```text
course_notes/
  01/
  02/
  legacy_projects/
```

Los notebooks de esta carpeta pueden conservar rutas historicas. La version
publicable y refactorizada de cada proyecto debe vivir en `projects/`.
