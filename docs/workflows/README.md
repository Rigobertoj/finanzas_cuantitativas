# Workflows

Guias operativas para ingesta de datos, construccion de datasets, ejecucion de
estrategias y generacion de reportes.

## Documentos

- `imports_and_environment.md`: configuracion de imports, instalacion editable y kernel de notebooks.
- `local_market_data_pipeline.md`: ingesta local multi-ticker con SQLite y manifest auditable.
- `market_data_ingestion.md`: uso operativo de providers ThetaData.
- `phase_2_summary_and_phase_3_thetadata.md`: cierre de contratos y preparacion del provider.
- `phase_3_summary_and_phase_4_storage_pipeline.md`: cierre de provider ThetaData y diseno de storage/pipeline local.
- `phase_4_summary_and_phase_5_hedging_pipeline.md`: cierre de storage SQLite y diseno del pipeline de hedging.
